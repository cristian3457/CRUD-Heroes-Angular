ACERCA DE CRUD HEROES

La aplicación CRUD Heroes se realizo con el fin de aplicar lo apredido en el curso de Angular de cero a experto de la plataforma Udemy.

Las principales funcionalidades y características que se aplicarón en este proyecto son:

  • 1.- Protección de rutas
  • 2.- Subida de imagenes
  • 3.- Creación de varios archivos de modulos y rutas
  • 4.- Formularios reactivos y uso de form builder
  • 5.- Uso de Local Storage
